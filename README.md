# 📊 MIS Automation Dashboard

> Automated MIS Reporting System built with Google Sheets, Google Apps Script & SQL that reduced manual reporting effort by **50%+** across operations and finance teams.

---

## 🎯 Problem Statement

Manual daily/weekly MIS reports were time-consuming and error-prone. Teams spent hours compiling data from multiple sources every day.

**Before Automation:**
- ❌ 2–3 hours daily for report generation
- ❌ Human errors in data entry
- ❌ Delayed insights for decision makers
- ❌ No automatic alerts for KPI exceptions

**After Automation:**
- ✅ Reports generated in minutes
- ✅ Zero manual data entry errors
- ✅ Real-time dashboards for stakeholders
- ✅ Automatic email alerts for exceptions

---

## 🛠️ Tools & Technologies

| Tool | Purpose |
|------|---------|
| Google Sheets | Data storage & pivot analysis |
| Google Apps Script | Automation & email alerts |
| SQL / BigQuery | Data extraction & querying |
| Power BI / Looker Studio | Dashboard visualization |
| Advanced Excel | KPI tracking & formatting |
| Power Query | Data transformation |

---

## 🚀 Key Features

- 📥 **Auto Data Pull** — Automatically pulls data from multiple sheets daily
- 📧 **Email Alerts** — Sends instant alerts when KPIs cross threshold
- 📊 **Weekly Summary** — Auto-generates weekly MIS summary report
- 🎨 **Color-coded Exceptions** — Visual highlighting for quick issue spotting
- ⏰ **Scheduled Triggers** — Runs automatically every day at 9 AM
- 🔍 **50,000+ Records** — Handles large datasets efficiently
